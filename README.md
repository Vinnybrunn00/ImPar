# PAR-OU-IMPAR-
Esse script verifica se o numero digitado é IMPAR ou PAR 

## INSTALANDO DEPENDENCIAS

```
pip install -r requerimentos.txt
```
## COMO EXECUTAR

```LINUX```

```
python3 Par_ou_impar.py
```

```WINDOWS```

```
./Par_ou_impar.py
```
