# PAR-OU-IMPAR-
This script checks if the number entered is ODD or PAR

## INSTALLING DEPENDENCIES

```
pip install -r requerimentos.txt
```
## HOW TO RUN

```LINUX```

```
python3 Par_ou_impar.py
```

```WINDOWS```

```
./Par_ou_impar.py
```
