# PAR-OU-IMPAR-
Esse script verifica se o numero digitado é IMPAR ou PAR 

## Instalando Dependências:

```Instalando o pyfiglet```
```
apt-get install pyfiglet
```

```Instalando o rich```
```
apt-get install rich
```
